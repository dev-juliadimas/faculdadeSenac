# 📄 Manipulação e Conversão de Dados (7 Projetos)

Esta pasta contém exercícios que exploram a **manipulação de *strings***, a **conversão entre tipos de dados** e a **formatação de saída**. São projetos cruciais para lidar com dados de forma eficaz.

## 🎯 Conceitos Chave

* **Strings e Concatenação:** Unir textos, números, ou partes de um nome (ex: gerar um nome de arquivo ou uma senha).
* **Conversão Complexa:** Uso do **operador módulo (`%`)** para converter segundos em horas, minutos e segundos.
* **Iteração de String:** Percorrer uma palavra caractere por caractere.

## 📁 Exemplos de Projetos

| Arquivo | Lógica Implementada |
| :--- | :--- |
| `seguntoToHoraMinSeg.java` | Converte segundos totais para o formato $\text{HH}:\text{MM}:\text{SS}$. |
| `arqAluno.java` | Calcula média e frequência, formatando o nome de um arquivo com os resultados. |
| `caracterPalavra.java` | Usa o laço `for` e `charAt()` para exibir cada caractere de uma palavra em uma linha. |
| `nomeFim.java` | Usa um laço para coletar nome/idade até que o nome seja "fim". |
| `tempoDownload.java` | Calcula o tempo de download, lidando com a conversão de MB para Mb. |
| **... (Adicionar o restante dos 2 arquivos)** | |

Para executar qualquer projeto, navegue até esta pasta no terminal e siga os passos (substitua pelo nome do seu arquivo):

cd manipulacaoDados/
